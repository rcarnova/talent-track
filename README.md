# Talent Track

Applicazione per la gestione e il tracciamento dei talenti.

## URL

- GitHub Pages: https://rcarnova.github.io/talent-track/
- Vercel: https://talent-track-blush.vercel.app

## Tecnologie utilizzate

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Come sviluppare in locale

Requisiti: Node.js & npm installati.

```sh
# Clona il repository
git clone https://github.com/rcarnova/talent-track.git

# Entra nella cartella
cd talent-track

# Installa le dipendenze
npm install

# Avvia il server di sviluppo
npm run dev
```
